An SVG Pie chart with tooltips

A straight-forward demo of a Pie chart with tooltips. Remember that by default the SVG highlights for tooltips are set with the pointer-events property set to none.

This goes in the documents header: 
<script src="RGraph.svg.common.core.js"></script>
<script src="RGraph.svg.common.tooltips.js"></script>
<script src="RGraph.svg.pie.js"></script>
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
Put this where you want the chart to show up: 
<div style="width: 650px; height: 350px" id="chart-container"></div>
This is the code that generates the chart: 
<script>
    new RGraph.SVG.Pie({
        id: 'chart-container',
        data: [85,48,13,56,79,95],
        options: {
            labels: ['Alf','Berty','Craig','Dean','Elgar','Fliss'],
            tooltips: ['Alf','Berty','Craig','Dean','Elgar','Fliss'],
            shadow: true,
            title: 'An SVG Pie chart with tooltips'
        }
    }).draw();
</script>
