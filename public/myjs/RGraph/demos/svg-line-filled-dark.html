An SVG Line chart using transparency

This is a Line chart that has some custom values applied to it. It's a filled chart, also using the filledAccumulative option too. Gradients have been given to the fills and different colours given to the strokes. And finally it's also using the trace effect too.

This goes in the documents header: 
<script src="RGraph.svg.common.core.js"></script>
<script src="RGraph.svg.line.js"></script>
Put this where you want the chart to show up: 
<div style="width: 750px; height: 300px; background-color: black" id="chart-container"></div>
