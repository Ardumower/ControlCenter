A basic SVG SemiCircular Progress chart

A very simple Semi-circular Progress meter. The meter can actally have more labels if required along with tooltips which can be used to provide more information to the user.

This goes in the documents header: 
<script src="RGraph.svg.common.core.js"></script>
<script src="RGraph.svg.semicircularprogress.js"></script>
Put this where you want the chart to show up: 
<div style="width: 450px; height: 250px" id="chart-container"></div>
This is the code that generates the chart: 
<script>
    new RGraph.SVG.SemiCircularProgress({
        id: 'chart-container',
        min: 0,
        max: 10,
        value: 7.8,
        options: {
            labelsCenterDecimals: 1
        }
    }).draw();
</script>
